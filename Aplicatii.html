Aplicatii

Aplicatia1

Avand in vedere o matrice binara, sortati-o in timp liniar si spatiu constant. Iesirea ar trebui sa imprime toate zerourile, urmate de toti unu.

De exemplu: Intrare: { 1, 0, 1, 0, 1, 0, 0, 1 } Iesire: { 0, 0, 0, 0, 1, 1, 1, 1 }

Ideea este sa folositi 1 ca element pivot si sa faceti o trecere a procesului de partitie. Matricea rezultata va fi sortata.

Aplicatia2

Avand in vedere o serie de numere intregi pozitive si negative, segregati-le in timp liniar si spatiu constant. Iesirea ar trebui sa imprime toate numerele negative, urmate de toate numerele pozitive.

De exemplu: Intrare: [9, -3, 5, -2, -8, -6, 1, 3] Iesire: [-3, -2, -8, -6, 5, 9, 1, 3 ]

Ideea este sa folositi 0 ca element pivot si sa faceti o trecere a procesului de partitie. Matricea rezultata va satisface constrangerile date.

Aplicatia3

Avand in vedere un tablou intreg, rearanjati-l astfel incat sa contina numere pozitive si negative in pozitii alternative. Daca matricea contine mai multe elemente pozitive sau negative, mutati-le la sfarsitul matricei.

De exemplu: Intrare: { 9, -3, 5, -2, -8, -6, 1, 3 } Iesire: { 5, -2, 9, -6, 1, -8, 3, -3 }

Ideea este sa folositi 0 ca element pivot si sa faceti o trecere a procesului de partitie. Matricea rezultata va contine toate numerele intregi pozitive pana la sfarsitul matricei si toate numerele intregi negative la inceput. Apoi schimbati elementele negative alternative de la urmatorul element pozitiv disponibil pana cand se ajunge la sfarsitul matricei sau toate numerele intregi negative sau pozitive sunt epuizate.

Aplicatia4

Având în vedere un tablou întreg, mutați toate zerourile prezente în ea până la sfârșit. Soluția ar trebui să mențină ordinea relativă a elementelor din matrice.

De exemplu: Intrare: { 6, 0, 8, 2, 3, 0, 4, 0, 1 } Ieșire: { 6, 8, 2, 3, 4, 1, 0, 0, 0 }

Ideea este sa folositi 0 ca element pivot si sa faceti o trecere a procesului de partitie. Logica de partitionare va citi toate elementele si de fiecare data cand intalnim un element non-pivot, il schimbam cu prima aparitie a pivotului.
